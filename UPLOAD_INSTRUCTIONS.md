# 📦 파일 업로드 지침서

## 개요
사장님이 직접 파일을 업로드하여 시스템에 적용할 수 있도록 준비된 파일들입니다.

## 업로드 파일 위치
- **업로드 대기 디렉토리**: `/home/user/webapp/upload-pending/`
- 이 디렉토리에 파일을 업로드하시면, 제가 검토 후 실제 프로젝트에 적용하겠습니다.

## 사용 방법

### 1️⃣ 파일 업로드
```bash
# 사장님이 파일을 업로드할 위치
/home/user/webapp/upload-pending/
```

### 2️⃣ 업로드 후 알림
업로드가 완료되면 저에게 알려주세요:
- "파일 업로드 완료했어요"
- "health-cert-upload.tsx 업로드했습니다"

### 3️⃣ 자동 적용
제가 업로드된 파일을 확인하고:
1. 파일 검증 (구문 오류, 타입 체크)
2. 적절한 위치로 이동
3. Git 커밋 및 푸시
4. 배포 확인

## 🔧 트러블슈팅

### 업로드 실패 시
1. **파일 크기 확인**: 5MB 이하인지 확인
2. **파일 형식 확인**: `.tsx`, `.ts`, `.json` 등 올바른 확장자인지 확인
3. **파일 이름 확인**: 한글이나 특수문자 없이 영문/숫자/하이픈만 사용

### 적용 실패 시
1. **구문 오류**: TypeScript 오류가 있는지 확인
2. **의존성 오류**: import 경로가 올바른지 확인
3. **타입 오류**: 타입 정의가 올바른지 확인
